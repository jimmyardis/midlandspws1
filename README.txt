MPWS Static Website (Demo)
==========================

What this is:
- A fast, mobile-friendly static site using your AI-generated branded images.
- Pages: Home, Services, About, Gallery, Contact.

How to run locally:
- Open index.html in a browser, or serve the folder:
  - Python: python -m http.server 8080

How to deploy:
- Upload the folder to Netlify, Vercel, Cloudflare Pages, or any web host.
- Connect the contact form to Formspree/Netlify Forms or your backend.

Replace / optimize images:
- Current images are large PNGs. For production, convert to WebP/JPG and compress.
